About Pincode Importer & Directory

⛓

Relational Extraction & Mapping

Automatically restructures flat Indian address CSV files into clean, relational database tables (States ➔ Districts ➔ Taluks ➔ Post Offices) maintaining full foreign key relationships.

🔍

Dynamic Schema Discovery

Queries database metadata on the fly. Works seamlessly with both singular or plural lookup schemas (e.g. app_state vs app_states) and maps correct ID columns automatically.

⚡

High-Performance Batches

Uses multithreaded processing and bulk insertions (batches of 2000) to safely load over 160,000 postal records over cloud connections in seconds without locking your system.

🗺

Interactive Geographic Search

Query pincodes instantly using a custom high-contrast zebra data table. Clicking any result pulls circle details, GPS coordinates, and provides direct links to locate them on Google Maps.

Quick Getting Started Guide

1

Database Setup

Go to the Import Directory tab and enter your MariaDB database credentials. Click Test Connection to verify access.

2

Specify CSV Source

Verify that your local Indian pincode directory CSV path is set correctly (e.g. all-india-pincode-html-csv-Ver-2026.csv).

3

Run Relational Import

Click Start Extraction & Import. Watch the scrolling live terminal logs as lookup caches are resolved and records are imported.

4

Search & Discover

Switch to the Search Pincode tab, input a 6-digit pincode, and select any post office record to display complete geographical data.
